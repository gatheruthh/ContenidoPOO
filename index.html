Ejercicio 1: Lógica Secuencial

Unidad I - Introducción

Conceptos Clave

  • • Paradigma Estructurado: Programación basada en flujo de control y funciones.

Guía Temática

Se refleja el tema de **Sintaxis** y **Manejo de variables locales**. El estudiante aprende la lógica base antes de la abstracción.

Ejercicio 2: Salto al Objeto

Unidad II - Clases

Conceptos de la Guía: Se aplican las definiciones de **Clase** y **Objeto**. Se observa la **Instanciación** en el Main: clsAdministrativos admon = new clsAdministrativos();. 

internal class clsAdministrativos {
    public float CalcularNominaAdmon(float sueldoBase, float isr) {
        return sueldoBase - (sueldoBase * isr / 100);
    }
}

internal class Program {
    static void Main(string[] args) {
        // Se crea el objeto (Instancia)
        clsAdministrativos admon = new clsAdministrativos();
        // Llamada al método a través del objeto
        float sueldo = admon.CalcularNominaAdmon(3000, 10);
    }
}

Ejercicio 3: Abstracción de Estado

Unidad II - Atributos

Abstracción

Mapeo: Determinar solo atributos necesarios. El objeto ya no solo hace algo, **ES** algo.

Reflejo Curricular

"Determinar los atributos y métodos que llevará el objeto".

internal class clsAdministrativos {
    public string nombreTrabajador;
    public float isr;
    public float sueldoBase;
    public float sueldoNeto;

    public float CalcularNominaAdmon() {
        // El método ya no pide parámetros, usa sus Atributos
        sueldoNeto = sueldoBase - (sueldoBase * isr / 100);
        return sueldoNeto;
    }
}

Ejercicio Final: Arquitectura Robusta

Unidad III - Buenas Prácticas
HERENCIA Uso de : clsTrabajadores para reducir redundancia.
ENCAPSULAMIENTO Uso de private y Property (get/set).
POLIMORFISMO Método abstract que cambia según el tipo de clase. 
// APLICACIÓN DE SOLID Y MODELADO AVANZADO
internal abstract class clsTrabajadores {
    public float isr;
    public string nombreTrabajador;
    protected float sueldoNeto; // Encapsulamiento PROTECTED
    
    // Propiedad (Buenas Prácticas Unidad III)
    public float SueldoNeto { get => sueldoNeto; set => sueldoNeto = value; }
    
    public abstract float CalcularNomina(); // Polimorfismo
}

internal class clsAdministrativos : clsTrabajadores {
    private float sueldoBase;
    public float SueldoBase { get => sueldoBase; set => sueldoBase = value; }

    public override float CalcularNomina() {
        sueldoNeto = SueldoBase - (SueldoBase * isr / 100);
        return sueldoNeto;
    }
}
